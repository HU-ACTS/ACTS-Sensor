Detailed Description

MPU9250 data read and setup.

This class handles the MPU9250 communication. It inherrits from the Sensor class and has the same methods.

Constructor & Destructor Documentation

◆ Mpu9250Implementation()

Mpu9250Implementation::Mpu9250Implementation ( )

Mpu9250Implementation constructor.

This method initializes all used states in this class. It also intializes the MPU9250 so that all data can be read with a single method (SensorRead).

Warning: I2C_NUM_0 is used and should be initialized before calling this constructor.
