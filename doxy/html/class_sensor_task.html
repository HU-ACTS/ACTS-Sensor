Detailed Description

Task that handles all sensor communication.

This task will sample at a frequency of SAMPLE_RATE_H data from all sensors (in this case BMP280 and MPU9250). A xTimer will run at that frequency and set a bit. The task will then read data from the sensors (which are connected through I2C) and send this data to the DoubleBuffer and DataProcessor.

If the sleepbit has been set this task will go into an infinite loop.
